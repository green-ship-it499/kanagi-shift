📖 使い方ガイド

Step 1：スタッフを登録

「スタッフ管理」タブでスタッフを登録してください。自動で token が生成されます。

Step 2：HTML を生成

スタッフカードの「📥 HTML生成」ボタンをクリック。スタッフ用の希望入力 HTML がダウンロードされます。

Step 3：スタッフに URL を配布

Netlify にアップロードした HTML の URL を各スタッフに LINE で送信してください。

Step 4：希望を集計

「希望集計」タブで月を選択すると、全スタッフの希望が一覧で表示されます。

Netlify へのデプロイ方法

  1. Netlify.com にアクセス
  2. ログイン（Google アカウントでもOK）
  3. スタッフ用 HTML ファイルをドラッグ&ドロップ
  4. 自動で URL が発行される
  5. その URL をスタッフに配布
⚠️ 注意： ブラウザの localStorage は、同じブラウザ・同じデバイスでのみデータが保存されます。スタッフが別のデバイスやブラウザで希望を入力した場合、データは同期されません。
